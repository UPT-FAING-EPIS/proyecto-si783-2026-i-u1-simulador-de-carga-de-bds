# Simulador de Carga de BD

Simula carga en bases de datos **SQL y NoSQL** directamente desde VS Code, sin necesidad de un servidor real.

## Motores soportados

- MySQL
- PostgreSQL
- SQL Server
- SQLite
- Oracle
- MongoDB
- Redis

## Cómo usar

1. Abre la paleta de comandos: `Ctrl+Shift+P`
2. Escribe **"Abrir Simulador de Carga de BD"**
3. Configura los parámetros y presiona **Iniciar Prueba**

## Características

- Usuarios virtuales configurables (10 – 500)
- Duración de prueba (30s – 600s)
- Tipos de consulta: SELECT, INSERT, UPDATE, DELETE
- **Comparar 2 motores** simultáneamente
- **Ataque progresivo** para encontrar el punto de saturación
- Métricas en tiempo real: TPS, latencia, CPU, conexiones
- Historial de pruebas
- Exportar reporte en PDF
